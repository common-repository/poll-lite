Allows to add polls to your site

== Description ==
With this plugin you can add one or more polls to your site. This can be done by pasting the shortcode of a poll in pages, posts or sidebar. For more features get the <a href="http://wordpress.phpanswer.com/poll/">Poll FULL</a> version.

== Installation ==
1. Upload `poll-lite x.y.z.zip` from the plugin panel or unzip the file and upload the folder `poll-lite` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` panel in WordPress
3. Go to `Settings->Poll` to set your preferences
4. Add polls from the `Polls` panel (in the left column of your admin page, like `Pages`, `Posts` etc.)
4. Paste the shortcode where you want to display the poll


If you get errors or it does not work, check the `poll-lite` plugin folder permissions. They should be 755.
Plugin requires php 5.2
